# Earth System Auditor

**Deskripsi:**
Aplikasi ini dibangun untuk membuktikan bahwa suhu di permukaan Bumi bukanlah hasil dari "anomali cuaca" yang acak, melainkan hasil dari perhitungan mekanis yang deterministik.

**Prinsip Kerja:**
1. **Input Absolut:** Menggunakan Konstanta Surya sebesar 1361 W/m².
2. **Reduksi Geometris:** Menghitung sudut datang foton berdasarkan Lintang dan Tanggal (Deklinasi).
3. **Reduksi Mekanis:** Menghitung kehilangan panas akibat penurunan tekanan udara (Lapse Rate 0.65°C/100m) menggunakan data elevasi otomatis.

**Tujuan:**
Membenahi pemahaman sistemik bahwa suhu adalah variabel yang dapat dihitung secara pasti melalui koordinat dan tekanan atmosfer.
